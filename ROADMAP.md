# CareBed Project Roadmap

A high-level plan for the CareBed system, organized by phases from MVP to advanced healthcare management features.

## Phase 1: Foundation & MVP

**Planned Start**: January 2026   
**Target Completion**: May 2026   
**Actual Completion**:

**Goal**: Establish core patient management, staff authentication, and basic care tracking.

- ### [ ] Project Setup
    - [x] Initialize repository and environment configuration
    - [x] Database schema design (Patients, Staff, Wards)
    - [ ] Establish API architecture with Go
- ### [ ] Authentication & Authorization
    - [x] Staff login and registration
    - [ ] Role-Based Access Control (Admin, Doctor, Nurse, Caretaker)
    - [ ] Secure session management
- ### [ ] Patient Management
    - [ ] Patient registration and profile creation
    - [ ] Digital medical charts and history
    - [ ] Admission and discharge workflow
    - [ ] Ward and bed assignment system
- ### [ ] Medication Tracking
    - [ ] Medication schedule creation
    - [ ] Dosage logging and verification
    - [ ] Real-time alerts for missed doses
- ### [ ] Dashboard & User Interface
    - [ ] Responsive dashboard for staff
    - [ ] Patient overview with vital signs and medication status
    - [ ] Basic reporting tools for patient care metrics
    - [ ] Real-time monitoring of patient vitals via dashboard (temperature, heart rate, etc.)
    - [ ] Automated alerts for critical conditions (e.g., high fever, low oxygen levels)
    - [ ] Integration with wearable health devices for continuous monitoring
### [ ] IoT Integration (Stretch Goal)
    - [ ] Smart bed sensor integration (weight, movement)
    - [ ] Real-time monitoring of patient vitals through connected devices
    - [ ] Automated alerts for abnormal readings
